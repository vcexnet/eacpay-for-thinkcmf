Eacpay支付插件使用帮助
安装插件并配置各个参数
加入hook配置 \app\portal\hooks.php
            'eacpay' => [
                "type"        => 3,
                "name"        => '调用Eacpay支付',
                "description" => "调用Eacpay支付",
                "once"        => 0
            ],
前台在需要使用eacpay的地方加上 
                <?php 
                    $order=array(
                        "out_trade_no"=>"订单号",
                        "amount"=>"金额",
                        "callback"=>"回调控制器,如:\app\portal\controller\ArticleController"
                    );
                ?>
                <hook name="eacpay" param="param"/>
